1.浮点数和定点数的概念

  • 定点数 = 用整数表示的浮点近似，约定小数点的位置不变

  • 假设你有一个小数 f，你想转成定点数 i，只需要：

  • $\textbf{i} = \text{round}(f \times 2^n)$

    其中：
    • f：原始浮点数（比如 0.625）
    • $2^n$：定点精度（小数点后 n 位）
    • i：整数表示的定点值
    • n：你选的定点位数，通常是 8、11、15、16

2.使用定点数计算的原因

  • GPU（尤其 OpenCL 的嵌入式后端，如 Adreno、Mali）上：
    • int 运算性能 >> float 运算
    • 定点数精度足够 + 更快
    • 减少功耗，适合移动端部署
    • OpenCL、嵌入式 GPU 通常选择 11~15 位
